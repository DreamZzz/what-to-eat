# 前端功能验证指南

## 验证入口

```bash
cd frontend
./start.sh local
```

## 建议验证链路

1. 注册 / 登录 / 找回密码
2. 首页帖子流
3. 搜索
4. 图文视频发布
5. 地点选择
6. 详情页与评论
7. 资料页与资料编辑

## 运行模式补充

- `./start.sh local`：模拟器 + 本地后端
- `./start.sh remote`：模拟器 + 远端后端
- `./start.sh device local "设备名"`：真机 + 本地后端
- `./start.sh device remote "设备名"`：真机 + 远端后端
